Fundamentos de Web Design I

Bem-vindo(a), nesse curso você irá aprender a construir páginas da web do zero, utilizando todos os recursos da linguagem HTML5 e a estilizar seu site com o poderoso CSS3 e todos atributos que há nele, assim te preparando para qualquer adversidade que encontrar no mercado de trabalho ou em qualquer projeto que tenha em mente.

Aula 01 - Introdução ao HTML

Nesta aula será ensinado o conceito de HTML, o conceito de tags, como elas funcionam e como devem ser estruturadas, qual a estrutura básica para construção de uma página da web com as tags <html>, <head> e <body> e como cada uma dessas tags funcionam, além de mostrar algumas das tags básicas para te auxiliar na construção de sua página.

Instrutor: Kelvin Arruda Silva Duração da aula: 7:36 Clique aqui para baixar o material de apoio desta aula
Questionário – Introdução ao HTML
  1. 1. O que é HTML?
    1. É uma linguagem de programação de computadores.
    2. É uma estrutura da língua portuguesa.
    3. É uma estrutura de marcação de texto para construção de páginas da web.
    4. É uma forma de marcação de instruções para o sistema operacional.
  2. 2. Quem criou o HTML e qual era seu objetivo?
    1. Foi criado por Steve Jobs para desenvolvimento da sua suíte office.
    2. Foi criado por Tim Berners-Lee e tinha como objetivo o compartilhamento de pesquisas com outros cientistas.
    3. Foi criado por Satya Nadella para criação de páginas para o Android.
    4. Foi criado por Bill Gates para desenvolvimento do Windows.
  3. 3. O que são Tags ?
    1. Tags são bibliotecas de C#.
    2. Tags são usados para declarar elementos em C++.
    3. Tags são elementos utilizados pelo usuário de navegadores para pesquisar algo na internet.
    4. Tags são elementos do HTML, elas são renderizadas pelo navegador para demonstrar um conteúdo. Ex. Textos, tabelas, imagens, etc.
  4. 4. Qual é a estrutura da maior parte das Tags ?
    1. <tag> conteúdo </tag>
    2. <tag atributos> conteúdo </tag>
    3. <tag>
    4. <tag atributos> conteúdo <tag/>
  5. 5. O que as tags <html>, <head>, <body>, respectivamente, significam ?
    1. Define o início e o fim de um documento em HTML. Define as metas informações sobre o documento HTML. Define o corpo do documento HTML e todas as informações visíveis ao usuário se encontram nessa tag.
    2. Define o corpo do documento HTML e todas as informações visíveis ao usuário se encontram nessa tag. Define as metas informações sobre o documento HTML. Define o início e o fim de um documento em HTML
    3. Define o início e o fim de um documento em HTML. Define o corpo do documento HTML todas as informações visíveis ao usuário se encontram nessa tag. Define as metas informações sobre o documento HTML.
    4. Nenhuma das alternativas anteriores.

Aula 02 - Boas práticas HTML

Nesta aula será ensinado algumas boas práticas a serem seguidas durante a construção de sua página da web, como ordenar corretamente os elementos HTML(tags), comentar o fechamento de cada div em seu código, identar seu código(e o que seria identação), declarar o tipo de codificação de caracteres de sua página, e utilizar tags opcionais.

Instrutor: Gledson Rodrigues de Oliveira Junior Duração da aula: 5:13 Clique aqui para baixar o material de apoio desta aula
Questionário – Boas práticas HTML
  1. 1. Qual dos seguintes códigos estão corretos?
    1. <p><a href="#">Site</p></a>
    2. <p><a href="#">Site</a></p>
    3. <p><a href="#">Site<a></p>
    4. <p><a href="#">Site</a><p>
  2. 2. Identar o código é:
    1. Todas as alternativas.
    2. Alinhar todo o código à direita.
    3. Fechar todas as tags que foram abertas.
    4. Organizar seu código para que ele fique legivel e de fácil entendimento.
  3. 3. Qual codificação recomendada para que todos caracteres sejam exibidos corretamente?
    1. UTF-8
    2. ISO-8859-1
    3. CHARSET
    4. META
  4. 4. Exemplos de tags opcionais que devem ser usadas são:
    1. <pre> <h6> <code>
    2. <h1> <title>
    3. <html> <head> <body>
    4. Todas as alternativas estão corretas.
  5. 5. É uma boa prática HTML:
    1. Comentar o fechamento das divs.
    2. Comentar todo o código.
    3. Fechar a tag só após inserir seu conteúdo.
    4. Inserir a tag body dentro da tag head.
