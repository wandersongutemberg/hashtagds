Fundamentos de Web Design I

Bem-vindo(a), nesse curso você irá aprender a construir páginas da web do zero, utilizando todos os recursos da linguagem HTML5 e a estilizar seu site com o poderoso CSS3 e todos atributos que há nele, assim te preparando para qualquer adversidade que encontrar no mercado de trabalho ou em qualquer projeto que tenha em mente.

Aula 01 - Introdução ao HTML

Nesta aula será ensinado o conceito de HTML, o conceito de tags, como elas funcionam e como devem ser estruturadas, qual a estrutura básica para construção de uma página da web com as tags <html>, <head> e <body> e como cada uma dessas tags funcionam, além de mostrar algumas das tags básicas para te auxiliar na construção de sua página.

Instrutor: Kelvin Arruda Silva Duração da aula: 7:36 Clique aqui para baixar o material de apoio desta aula
Questionário – Introdução ao HTML
  1. O que é HTML?
  2. Quem criou o HTML e qual era seu objetivo?
  3. O que são Tags ?
  4. Qual é a estrutura da maior parte das Tags ?
  5. O que as tags <html>, <head>, <body>, respectivamente, significam ?

Aula 02 - Boas práticas HTML

Nesta aula será ensinado algumas boas práticas a serem seguidas durante a construção de sua página da web, como ordenar corretamente os elementos HTML(tags), comentar o fechamento de cada div em seu código, identar seu código(e o que seria identação), declarar o tipo de codificação de caracteres de sua página, e utilizar tags opcionais.
