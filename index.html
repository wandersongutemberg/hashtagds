Olá! Bem vindo(s) a #.NET
sua escola digital!

Aqui você encontra os melhores cursos em vídeo das disciplinas de Lógica de Programação, Fundamentos de Web Design I, Sistemas Operacionais, Arquitetura e Organização de Computadores, Inglês Instrumental e Metodologia Cietífica. Além das vídeos aulas temos também questionários para você testar seus conhecimentos adquiridos e você pode entrar em contato conosco sempre que quiser tirar alguma dúvida.

CURSOS OFERECIDOS

Lógica de Programação

Nesse curso você irá aprender toda a lógica de programação utilizando a linguagem Java, uma linguagem de programação orientada a objetos que possibilita o desenvolvimento para Web, Mobile e Desktop, e um mercado consolidado.

Começar agora!

Fundamentos de Web Design I

Nesse curso você irá aprender a criar sites utilizando o uso da linguagem HTML5 e a estilizar os mesmo com os recursos da linguagem CSS3, conhecimentos fundamentais para quem quer se tornar um desenvolvedor front-end de sucesso.

Começar agora!

Sistemas Operacionais

Nesse curso você irá adquirir conhecimentos sobre sistemas operacionais, aprender sobre a instalação e configuração de sistema operacional, firewall, monitorar o uso de memória do computador, criar e executar scripts, e muito mais.

Começar agora!

Arquitetura e Organização de Computadores

Nesse curso você irá aprender sobre sistemas digitais para viabilizar o entendimento dos sistemas computacionais. As partes básicas que compõem os computadores pessoais e como elas interagem entre si, considerando seu funcionamento e desempenho.

Começar agora!

Inglês Instrumental

Nesse curso você irá aprender a ler e compreender adequadamente a língua inglesa em textos relacionados à área de Sistemas para Internet e informática. Ler e compreender de forma autônoma textos e gêneros diversos, relacionados à área de Informática.

Começar agora!

Metodologia Científica

Nesse curso você irá aprender e compreender a metodologia científica para o planejamento, execução, análise e interpretação de pesquisa científica, com uso aplicado na área de informática, visando se tornar um profissional completo.

Começar agora!
